# Workspace Rules

- **No In-Text Emojis Rule**: Do NOT use in-text emoji icons in string literals (e.g. do not embed emojis like 📊, 📦, 💰, ⚡, ✨, ✏️, ➕, 💎, 👔, 📝, 👁️, 👑, 🗑️, ❌, ⚠️ in labels, titles, placeholders, toasts, or text nodes). Always keep text clean and use Lucide SVG icon components when visual icons are needed.
